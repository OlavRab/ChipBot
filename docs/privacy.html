Privacy Policy

Last updated: June 2026

This Privacy Policy explains what information ChipBot collects, how it is used, and how it is stored. By using ChipBot you agree to the practices described here.

1. Who We Are

ChipBot is operated by an individual developer. You can contact us at olav@element6consulting.nl with any privacy-related questions.

2. What Data We Collect

Data Why we collect it Where it's stored
Discord Server ID To associate uploaded sounds with the correct server Server filesystem / database
Discord User ID & username To attribute sound plays to users in usage logs; to verify server membership during OAuth login SQLite database on server
Uploaded .mp3 files Core service functionality — playing sounds in voice channels Server filesystem, organised by server ID
Sound play events Usage analytics (which sounds are played, by whom, in which server) SQLite database on server
Discord OAuth access token To fetch your server list so you can manage sounds on the website Server-side session only — never persisted to disk

We do not collect message content, voice data, personal addresses, payment information, or any data beyond what is listed above.

3. How We Use Your Data

  • To store and serve your uploaded sounds within Discord voice channels
  • To enforce per-server upload limits
  • To attribute sound play events in usage logs for analytics purposes
  • To verify Discord server membership when you use the upload website

We do not sell, trade, or share your data with third parties, except as required by law.

4. Data Retention

Sound files are stored for as long as the bot remains in your server. When ChipBot is removed from a server, all associated sound files are deleted. Usage logs are retained for operational purposes and reviewed periodically.

OAuth session data (your Discord access token and guild list) is held only in your browser session and is discarded when you log out or your session expires.

5. Third-Party Services

ChipBot interacts with the following third-party services:

  • Discord — the platform the bot operates on. Discord's own Privacy Policy applies to your use of their platform.
  • Revolut — used to process one-time upgrade payments. If you choose to upgrade your server tier, your Discord Server ID is included in the payment form so we can apply the upgrade. Revolut's own Privacy Policy applies to payment data.
  • CoinCap API — used for the /crypto command. No personal data is sent; only a coin name is included in the request.
  • JokeAPI — used for the /joke command. No personal data is sent.
  • Open Notify API — used for the /space command. No personal data is sent.

6. Your Rights

You may request deletion of your data at any time by contacting us at olav@element6consulting.nl. For sound files, you can also delete them directly via the upload website or using the /rmchip Discord command (requires Manage Server permission).

7. Security

We take reasonable technical measures to protect stored data, including path validation on all file operations and token-based authentication for administrative endpoints. However, no system is perfectly secure, and we cannot guarantee absolute security.

8. Children's Privacy

ChipBot is not directed at children under 13. We do not knowingly collect data from children under 13. If you believe a child has provided us with personal data, please contact us and we will delete it.

9. Changes to This Policy

We may update this Privacy Policy from time to time. The date at the top of this page reflects the most recent revision. Continued use of the service after changes constitutes acceptance of the updated policy.

10. Contact

Privacy questions or data deletion requests: olav@element6consulting.nl
